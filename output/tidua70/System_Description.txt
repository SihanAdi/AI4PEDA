1 System Description This TI design™ demonstrates the integration of the SimpleLink CC3200 Wireless MCU and the TI NFC device (DLP-7970A). The main use case sends an email to a preconfigured email server using an SMTP email application when the NFC reader detects a TAG. The MCU in the system is the CC3200 that controls the NFC device over an SPI interface The system starts at configuration mode that lets you provision the SimpleLink-based system to the access point. The current application supports the following two ISO types: • ISO15693 • ISO14443A Figure 1. System Description
